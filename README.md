## CLGrid
一个简易表格的控件，支持列头浮动与行头浮动以及合并列头行头单元格

### 环境
* build : xCode 6.2
* test run : ios7,ios8

### 使用方法
* 1. 把 `CLGridDemo/CLGrid`整个拖入项目
* 2. 引入头文件:`CLGrid.h`
